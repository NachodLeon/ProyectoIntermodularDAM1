
## Cómo usar

1. Abre **index.html** en tu navegador
2. Haz clic en **"Entrar"** para ver la galería
3. Desde la galería puedes:
   - **Buscar** vehículos
   - **Cambiar el tema** (Claro/Oscuro/Personalizado)
   - **Añadir** nuevas fotos
   - **Pedir cita** haciendo clic en cualquier tarjeta o en el botón rojo

## Funcionalidades

| Función | Cómo funciona |
|---------|----------------|
| Buscador | Filtra vehículos por título |
| Temas | Cambia colores claro/oscuro o elige los tuyos |
| Añadir foto | Sube imágenes desde tu ordenador |
| Pedir cita | Rellena el formulario y se guarda automáticamente |
| Tarjetas clickables | Al hacer clic en un vehículo se abre el formulario de cita |

## Datos guardados
- Las fotos añadidas se muestran pero **no se guardan** al recargar
- Las citas se guardan en el navegador (localStorage)

## Requisitos
- Navegador actual (Chrome, Firefox, Edge, Safari)
- Conexión a internet (para las imágenes por defecto)
