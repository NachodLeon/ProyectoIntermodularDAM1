## Descripción general del proyecto.

Este proyecto consiste en el desarrollo completo de una aplicación para la gestión completa de un concesionario de coches.

La aplicación permite:
- Gestionar el catálogo de coches (CRUD completo)
- Gestionar clientes
- Registrar ventas
- Programar citas en el taller



## ¿Qué problema resuelve?

Permite tener un control de citas, clientes, trabajadores y vehículos de manera que no haga falta tenerlo en papel y el administrador pueda gestionar y acceder a toda la información.



## ¿Para quién está pensado?

- Para un taller que también hace ventas de vehículos



## ¿Qué tecnologías hemos utilizado?

SQL
