# PROYECTO INTERMODULAR – GESTIÓN DE CONCESIONARIO

## Descripción General

Este proyecto consiste en el desarrollo completo de una aplicación para la **gestión integral de un concesionario de vehículos**. La aplicación permite gestionar el inventario de coches y motos, registrar ventas, administrar clientes y programar citas en el taller.

El proyecto abarca tres áreas principales:
- **Programación (Java + JDBC)** - Aplicación de escritorio
- **Base de Datos (MySQL)** - Almacenamiento de datos
- **Lenguaje de Marcas (HTML/CSS/JS)** - Interfaz web

---

## ¿Qué problema resuelve?

Permite tener un control centralizado de:
- Catálogo de vehículos
- Clientes registrados
- Ventas realizadas
- Citas del taller
- Mecánicos y sus especialidades

Eliminando la necesidad de gestionar esta información en papel y permitiendo al administrador acceder y gestionar todos los datos desde una sola aplicación.

---

## ¿Para quién está pensado?

- **Concesionarios de vehículos** que también tienen taller mecánico
- **Pequeños talleres** que venden vehículos de segunda mano
- **Administradores** que necesitan controlar ventas y citas
